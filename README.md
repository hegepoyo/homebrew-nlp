# homebrew-nlp

a Homebrew keg that specialized in Natural Language Processing.

自然言語処理に関連するライブラリに特化したHomebrewリポジトリ

## Install
```bash
$ brew tap oame/nlp
```

## Package index
- Julius(julius)
- Julius Dictation Kit(julius-dictation-kit)
- HTS Engine(hts_engine)
- Open JTalk(open_jtalk)
